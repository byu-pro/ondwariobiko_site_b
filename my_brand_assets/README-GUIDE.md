# My Brand Assets - Upload Guide

Welcome! This folder acts as your drop-zone for all the custom visual assets we'll need to fully personalize your new portfolio template. 

Here is what you need to upload into these folders so I can implement them perfectly for you:

### 1. `\logos`
Please drop your desired logos here.
- Ideally, include both a light-colored version (for dark mode) and a dark-colored version (for light mode).
- Example: `logo-dark.png`, `logo-light.png`, or `logo.svg`

### 2. `\icons`
This is for your favicons and browser tab icons.
- If you have an Apple Touch Icon, Android Chrome Icons, or a standard ICO file, put them here.
- A high-resolution `favicon.png` (like the one we just used) is perfect!

### 3. `\profile_photos`
The template requires some high-quality portrait/personal photos of you.
- We need one large vertical portrait image for the Hero Section (next to your stats).
- We also need one for the "About Me" page. 
- Example: `hero_portrait.jpg`, `about_me_photo.png`

### 4. `\portfolio_images`
This will be used whenever we update the "Projects" section or specific case studies. 
- High-quality rectangular or square thumbnails.
- Just drop your best project images here, and name them clearly (e.g., `project-nike-app.png`).

---

**Next Steps:**
Once you place a file into any of these folders, just let me know (e.g., *"I've uploaded my new portrait to the profile_photos folder"*), and I'll immediately compress it, optimize it, and seamlessly integrate it into the template code for you!
